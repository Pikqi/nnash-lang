### Jezik NNASH

NNash is not a shell

NNash je pisan u jeziku zig 0.15.2


Pokretanje sa `zig build run -- <path_do_fajla>`

Testni fajlovi se nalaze u `src/test_files/` u folderima `lexical_error`, `parsable` i `parsing_error`.
Ovi testni fajlovi su takodje pokriveni testovima koji su napisani u src/testing.zig
